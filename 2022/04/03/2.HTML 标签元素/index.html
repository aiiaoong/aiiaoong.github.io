10、列表

  • 有序列表
  • 自定义列表

（1）无序列表 Unordered List

列表项 List Item

（2）有序列表 Ordered List

（3）自定义列表 Definition List

标签含义

  • dl是definition lists的英文缩写 (自定义列表)
  • dt是definition term的缩写 (自定义列表组)
  • dd是definition description的缩写（自定义列表描述）

11、表格

（1）基本元素

标签含义

  • tr Table Row
  • th Table Header
  • td Table Data

table 属性：

  • border 边框宽度
  • width 表格宽度
  • height 表格高度

（2）表格结构，可以省略

  • thead 表格头部
  • tbody 表格主体
  • tfoot 表格底部

（3）合并单元格

  • 跨行合并（垂直）rowspan
  • 跨列合并（水平）colspan

左上原则

  • 上下合并，保留最上
  • 左右合并，保留最左

Tips: 不能跨结构合并
